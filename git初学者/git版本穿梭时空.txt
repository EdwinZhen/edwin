|  重返未来：							|
|								|
|	先用  git reflog 找到要回去的commit_id ，		|
|								|
|	命令  git reset --hard commit_id。			|
|								|
|---------------------------------------------------------------|
|								|
|  版本回退，不可提交，本地查看主要手段：			|
|								|
|  $ git reset --hard HEAD~3					|
|								|
|  本质上就是： git  reset 版本号。				|
|								|
|---------------------------------------------------------------|
|								|
|  版本回退，可提交。远程可更新：				|
|								|
|  revert指定版本可以实现版本回退，会创建一个新的提交		|
|								|
|  $ git revert --no-commit HEAD~2^..HEAD			|
|								|
|  不可：  git  revert 版本号。					|
